Task 1: Exploring and Visualizing the Iris Dataset¶

Objective¶

The goal of this task is to load, inspect, analyze, and visualize the Iris dataset using Python libraries such as Pandas, Matplotlib, and Seaborn.

Dataset¶

Iris Dataset from Seaborn.

Scatter Plot¶

Relationship between sepal length and petal length.

In [6]:
plt.figure(figsize=(8,6))

sns.scatterplot(
    data=iris,
    x="sepal_length",
    y="petal_length",
    hue="species"
)

plt.title("Sepal Length vs Petal Length")
plt.show()

Box Plot¶

Used to identify outliers.

In [8]:
plt.figure(figsize=(10,6))

sns.boxplot(data=iris)

plt.title("Box Plot of Iris Features")
plt.show()

Findings¶

  1. Setosa flowers are clearly separated from other species.
  2. Petal dimensions are highly useful for classification.
  3. Very few outliers exist in the dataset.
  4. Data appears well balanced.

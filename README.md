# AI/ML Internship Tasks

This repository contains solutions for the AI & ML Internship tasks. The project demonstrates data analysis, visualization, classification, and regression techniques using Python and Machine Learning libraries.

### Objective
Explore and visualize the Iris dataset using Python.

### Dataset
Iris Dataset

### Tools
- Python
- Pandas
- Seaborn
- Matplotlib

### Work Done
- Data Loading
- Data Inspection
- Summary Statistics
